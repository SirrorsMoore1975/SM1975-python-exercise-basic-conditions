# say_hellos

This question is an advanced question and may required deeper understanding on the Python programming

## Introduction

There are approximately 192 countries, each of which has their own or similar languages to one another,

## Pre-requirement

For this exercise you needed to know the following:

- How to write `Classes` in Python
- How to interact with a `json` file in Python

## Task

You task is to write a class that can:

### the `hellos` languages

- handle add / remove `hellos` language
- handle change / amended the `hellos` of a given language, so that changes do reflected upon greeting the person
- allow a particular `hello` language to reset to default

### the age limit

- check the age limit of that particular language speaker
- use given age property to see if this person is at the age limit or above (i.e. the default age, `20`, to became an adult)
- have a function that can alter the age limit
- have a function to alter a particular languages speaker age limit, and allow reset to default age limit for that language
- allow age limited to reset to default, i.e. 20

### the consolidate function

- handle greeting to a given user, containing its name, age, and language speak
- read to a `json` file, and will not alter the content of the given `json` file, this `json` file data should be consider the language default
- handle a full reset function so that the `hellos` language, age limit return to its default value
